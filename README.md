## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/MemerGamer/zenager.git
   cd zenager
   ```

2. Install Node.js using nvm:

   ```bash
   nvm install
   nvm use
   ```

3. Install dependencies with Bun:

   ```bash
   bun install
   ```

4. Start the development server:

   ```bash
   bun dev
   ```
